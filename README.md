# Path Tracer
Modern path-tracer.

![Default scene ray-traced and shaded using diffuse shading model](README.png)

## How to run

### Prerequisities
1. [CMake](https://cmake.org/)
2. [Connan package manager](https://conan.io/)

### Build and run
'
mkdir .conan && cd .conan
conan install .. -s compiler.libcxx=libstdc++11
'
